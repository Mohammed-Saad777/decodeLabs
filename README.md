# DecodeLabs - Award-Winning Landing Page

A modern, professional, fully responsive landing page built with pure HTML5, CSS3, and Vanilla JavaScript. No frameworks, no dependencies - just clean, performant code.

## \ud83c\udf1f Features

- **Award-Winning Design**: Premium, minimal aesthetic inspired by Apple, Stripe, and Vercel
- **100% Responsive**: Perfect on all devices - desktop, tablet, and mobile
- **Pure Vanilla JavaScript**: No React, jQuery, or any framework dependencies
- **Modern Animations**: Smooth scroll reveals, parallax effects, and micro-interactions
- **Performance Optimized**: Fast loading with efficient code
- **Accessibility**: Semantic HTML5 and ARIA labels
- **SEO Ready**: Proper heading hierarchy and meta tags
- **Cross-Browser Compatible**: Works on all modern browsers

## \ud83d\udc68\u200d\ud83d\udcbb Development

### Customization
All colors, fonts, and spacing are defined in CSS variables at the top of `style.css`:

```css
:root {
    --primary: #2563EB;
    --secondary: #0F172A;
    --accent: #3B82F6;
    /* ... more variables */
}
```

### Adding New Sections
1. Add HTML in `index.html`
2. Style in `css/style.css`
3. Add responsive rules in `css/responsive.css`
4. Add interactions in `js/script.js`
